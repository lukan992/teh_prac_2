# Limitations

1. Синтетические данные не доказывают абсолютную применимость методов в реальных условиях наблюдения БПЛА.
2. Все методы реализованы как адаптированные версии ключевых идей статей, а не как полные реплики оригинальных пайплайнов.
3. Point cloud методы проверяются в упрощенном сценарии с синтетически заданной геометрией сцены.
4. ResNet + HDBSCAN рассматривается как основной пользовательский метод сравнения, а не как метод из предыдущей практики.
5. Методы разных классов задач нельзя сравнивать одной метрикой без контекстного пояснения.
6. TRACLUS, Vector Field k-Means, sparse point cloud estimation, Cluster Filter и CL-Det реализованы в упрощенном adapted виде.
7. Embedding-метод использует 2D визуальное представление траектории вместо полноценной 3D ResNet архитектуры.
